# TSK-09 환율 사용자 매뉴얼

## 1. 개요

환율 위젯은 주요 통화의 환율 정보를 실시간으로 표시한다.

## 2. 표시 환율

| 통화쌍 | 설명 |
|--------|------|
| USD/KRW | 미국 달러 / 한국 원 |
| JPY/KRW | 일본 엔 / 한국 원 |
| EUR/KRW | 유로 / 한국 원 |
| CNY/KRW | 중국 위안 / 한국 원 |

## 3. 표시 정보

- **환율**: 현재 환율 (예: ₩1,450.25)
- **전일 대비**: 등락 금액 및 비율
- **고가/저가**: 당일 최고/최저 환율

## 4. 색상 규칙

- **상승 (빨간색)**: 환율 상승 (원화 약세)
- **하락 (파란색)**: 환율 하락 (원화 강세)

## 5. 데이터 갱신

- 자동 갱신 주기: 60초
- 데이터 소스: ExchangeRate-API (open.er-api.com)

## 6. 다른 위젯 연동

환율 데이터는 다음 위젯에서 KRW 환산에 활용된다:
- TSK-08 귀금속: USD 가격의 KRW 환산
- TSK-06 해외 주식: USD 가격 참고
