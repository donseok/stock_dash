# TSK-05 국내 주식 사용자 매뉴얼

## 1. 개요

국내 주식 위젯은 한국 주식시장(KRX)의 주요 종목 시세를 실시간으로 표시한다.

## 2. 표시 종목

| 종목코드 | 종목명 | 시장 |
|----------|--------|------|
| 005930 | 삼성전자 | KOSPI |
| 247540 | 에코프로 | KOSDAQ |
| 068270 | 셀트리온 | KOSPI |

## 3. 표시 정보

각 종목 카드에 다음 정보가 표시된다:
- **종목명 / 종목코드**: 삼성전자 (005930)
- **현재가**: 원화 표시 (예: ₩79,000)
- **전일 대비**: 등락금액 및 등락률 (예: +₩1,200 (+1.54%))
- **거래량**: 당일 거래량

## 4. 색상 규칙

한국 시장 관례에 따라:
- **상승 (빨간색)**: 전일 종가 대비 상승
- **하락 (파란색)**: 전일 종가 대비 하락
- **보합 (회색)**: 변동 없음
