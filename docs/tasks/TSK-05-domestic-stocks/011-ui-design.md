## 3. 컴포넌트 상세

### 3.1 WidgetHeader

| 요소 | 설명 | 스타일 |
|------|------|--------|
| 제목 | "국내 주식" | `text-lg font-semibold text-gray-900` |
| 새로고침 버튼 | 수동 갱신 | `text-gray-400 hover:text-gray-600` |

### 3.2 StockCard

| 요소 | 설명 | 스타일 |
|------|------|--------|
| 종목명 | 한글 종목명 | `text-base font-medium` |
| 종목 코드 | 숫자 코드 | `text-sm text-gray-500` |
| 시장 구분 | KOSPI/KOSDAQ | `text-xs px-2 py-0.5 rounded bg-gray-100` |
| 현재가 | 원화 가격 | `text-xl font-bold` |
| 등락 금액 | 전일 대비 변동 | 상승: `text-red-500`, 하락: `text-blue-500` |
| 등락률 | 퍼센트 | 등락 금액과 동일 색상 |
| 거래량 | 당일 거래량 | `text-sm text-gray-500` |
| 등락 화살표 | ▲ / ▼ | 상승: `text-red-500`, 하락: `text-blue-500` |

## 6. 인터랙션

### 6.1 자동 갱신

- 30초마다 자동으로 데이터 갱신
- 갱신 중 로딩 인디케이터 표시 (카드 우측 상단 spinning icon)
- 업데이트 시간 표시 갱신

### 6.2 수동 갱신

- 새로고침 버튼 클릭 시 즉시 데이터 갱신
- 갱신 중 버튼 비활성화 (중복 요청 방지)

### 6.3 카드 클릭

- 종목 카드 클릭 시 `onStockClick(symbol)` 콜백 실행
- 주식 차트 (TSK-04) 위젯과 연동하여 해당 종목 차트 표시
