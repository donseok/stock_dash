# TSK-10 시장 지수 사용자 매뉴얼

## 1. 개요

시장 지수 바는 대시보드 상단에 위치하여 국내외 주요 시장 지수를 한눈에 보여준다.

## 2. 표시 지수

### 2.1 국내 지수

| 지수명 | 심볼 | 설명 |
|--------|------|------|
| KOSPI | ^KS11 | 한국거래소 유가증권시장 종합지수 |
| KOSDAQ | ^KQ11 | 한국거래소 코스닥시장 종합지수 |

### 2.2 해외 지수

| 지수명 | 심볼 | 설명 |
|--------|------|------|
| DOW | ^DJI | 다우존스 산업평균지수 |
| NASDAQ | ^IXIC | 나스닥 종합지수 |

## 3. 표시 정보

각 지수 배지에 표시되는 정보:
- **지수명**: KOSPI, KOSDAQ, DOW, NASDAQ
- **현재값**: 소수점 2자리 (예: 2,650.25)
- **등락**: 전일 대비 변동값 및 변동률

## 4. 색상 규칙

### 4.1 국내 지수 (한국식)
- **상승 (빨간색)**: 전일 대비 상승
- **하락 (파란색)**: 전일 대비 하락

### 4.2 해외 지수 (국제식)
- **상승 (초록색)**: 전일 대비 상승
- **하락 (빨간색)**: 전일 대비 하락

## 5. 데이터 갱신

- 자동 갱신 주기: 10초
- 데이터 소스: Yahoo Finance
- 시장 마감 시 마지막 종가 표시
