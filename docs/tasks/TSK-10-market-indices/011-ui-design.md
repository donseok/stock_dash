## 3. 컴포넌트 상세

### 3.1 MarketIndicesBar

| 요소 | 설명 | 스타일 |
|------|------|--------|
| 컨테이너 | 상단 인디케이터 바 | `flex items-center gap-4 bg-white border-b px-4 py-2 overflow-x-auto` |
| 국내/해외 구분선 | 세로 구분선 | `border-l border-gray-300 h-8 mx-2` |

### 3.2 IndexBadge

| 요소 | 설명 | 스타일 |
|------|------|--------|
| 지수명 | KOSPI, KOSDAQ, DOW, NASDAQ | `text-xs font-semibold text-gray-600 uppercase` |
| 지수값 | 현재 포인트 | `text-sm font-bold` |
| 등락 포인트 | 전일 대비 변동 | `text-xs` |
| 등락률 | 퍼센트 | `text-xs` |
| 화살표 | ▲ / ▼ | 등락 방향 |

### 3.3 색상 규칙

#### 국내 지수 (KOSPI, KOSDAQ)

| 상태 | 색상 | 비고 |
|------|------|------|
| 상승 | `text-red-500` | 한국식 상승=빨강 |
| 하락 | `text-blue-500` | 한국식 하락=파랑 |
| 보합 | `text-gray-500` | 변동 없음 |

#### 해외 지수 (DOW, NASDAQ)

| 상태 | 색상 | 비고 |
|------|------|------|
| 상승 | `text-green-500` | 국제 표준 상승=초록 |
| 하락 | `text-red-500` | 국제 표준 하락=빨강 |
| 보합 | `text-gray-500` | 변동 없음 |

## 4. 반응형 디자인

| 화면 크기 | 레이아웃 | 비고 |
|-----------|---------|------|
| Desktop (>=1280px) | 4개 배지 한 줄 표시 | 전체 표시 |
| Desktop (1024-1279px) | 4개 배지 한 줄, 축소 | 등락률 생략 가능 |
| Tablet (768-1023px) | 가로 스크롤 | 스크롤 힌트 표시 |
| Mobile (<768px) | 가로 스크롤 | 2개 정도 보이고 스크롤 |

## 5. 인터랙션

### 5.1 자동 갱신

- 30초 간격 자동 갱신
- 갱신 시 변동된 지수에 flash 애니메이션

### 5.2 배지 호버

```css
.index-badge:hover {
  background-color: #f9fafb;
  cursor: pointer;
}
```

- 호버 시 툴팁으로 추가 정보 표시 (전일 종가, 시가, 거래량 등)

### 5.3 배지 클릭

- 클릭 시 해당 지수 상세 정보 또는 차트 표시

## 6. 접근성 (A11y)

| 항목 | 구현 |
|------|------|
| 스크린 리더 | `aria-label="코스피 2,650.25 포인트, 15.30 포인트 상승"` |
| 키보드 | Tab으로 배지 이동, Enter로 상세 표시 |
| 스크롤 | `role="tablist"` 시맨틱 마크업 |
| 라이브 업데이트 | `aria-live="polite"` 지수 변동 알림 |
