## 3. 컴포넌트 상세

### 3.1 CryptoCard

| 요소 | 설명 | 스타일 |
|------|------|--------|
| 코인 아이콘 | 심볼 배지 | `w-8 h-8 rounded-full bg-orange-100` (BTC), 코인별 색상 |
| 코인명 | 영문 이름 | `text-base font-medium` |
| 심볼 | BTC 등 | `text-sm text-gray-500` |
| 현재가 | KRW 가격 | `text-xl font-bold` |
| 24h 변동 | 금액/퍼센트 | 상승: `text-red-500`, 하락: `text-blue-500` |
| 24h 거래량 | KRW 거래량 | `text-sm text-gray-500` |

### 3.2 코인별 아이콘 색상

| 코인 | 배지 배경색 | 텍스트 색상 |
|------|-----------|-----------|
| BTC | `bg-orange-100` | `text-orange-600` |
| ETH | `bg-indigo-100` | `text-indigo-600` |
| SOL | `bg-purple-100` | `text-purple-600` |
| XRP | `bg-gray-100` | `text-gray-600` |
| LINK | `bg-blue-100` | `text-blue-600` |
| SUI | `bg-cyan-100` | `text-cyan-600` |
| ONDO | `bg-slate-100` | `text-slate-600` |

### 3.3 카드 레이아웃

```
┌──────────────────────────────────────┐
│ [아이콘]  코인명            심볼     │
│           ₩가격     ▲ 변동(변동률)  │
│           24h Vol: ₩거래량           │
└──────────────────────────────────────┘
```

- 카드 간격: `gap-2` (8px)
- 카드 패딩: `px-4 py-3`
- 구분선: `border-b border-gray-100` (마지막 제외)

## 4. 색상 체계

| 상태 | 텍스트 색상 | 비고 |
|------|-----------|------|
| 상승 | `#ef4444` (red-500) | 한국식 상승=빨강 (KRW 마켓) |
| 하락 | `#3b82f6` (blue-500) | 한국식 하락=파랑 |
| 보합 | `#6b7280` (gray-500) | 변동 없음 |

| 화면 크기 | 레이아웃 | 카드 표시 |

## 6. 인터랙션

### 6.1 자동 갱신

- 10초 간격 자동 갱신 (암호화폐 24/7 거래 특성)
- 가격 변동 시 숫자 하이라이트 애니메이션 (flash 효과, 300ms)

### 6.2 가격 변동 애니메이션

```css
@keyframes price-flash-up {
  0% { background-color: rgba(239, 68, 68, 0.2); }
  100% { background-color: transparent; }
}
@keyframes price-flash-down {
  0% { background-color: rgba(59, 130, 246, 0.2); }
  100% { background-color: transparent; }
}
```

### 6.3 카드 클릭

- 카드 클릭 시 해당 암호화폐 상세 / 차트 표시

## 7. 접근성 (A11y)

| 항목 | 구현 |
