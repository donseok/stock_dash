# TSK-07 암호화폐 사용자 매뉴얼

## 1. 개요

암호화폐 위젯은 Upbit 거래소의 주요 암호화폐 실시간 시세를 KRW 기준으로 표시한다.

## 2. 표시 종목

| 심볼 | 종목명 | 마켓 |
|------|--------|------|
| BTC | 비트코인 | KRW-BTC |
| ETH | 이더리움 | KRW-ETH |
| SOL | 솔라나 | KRW-SOL |
| XRP | 리플 | KRW-XRP |
| LINK | 체인링크 | KRW-LINK |
| SUI | 수이 | KRW-SUI |
| ONDO | 온도 | KRW-ONDO |

## 3. 표시 정보

- **종목명 / 심볼**: 비트코인 (BTC)
- **현재가**: KRW 표시 (예: ₩145,000,000)
- **24시간 등락**: 금액 및 비율
- **24시간 거래량**: 코인 수량

## 4. 색상 규칙

한국 시장 관례에 따라:
- **상승 (빨간색)**: 24시간 전 대비 상승
- **하락 (파란색)**: 24시간 전 대비 하락

## 5. 데이터 갱신

- 자동 갱신 주기: 10초
- 24시간 365일 운영
- 데이터 소스: Upbit REST API
