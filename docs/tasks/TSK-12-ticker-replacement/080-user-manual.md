# TSK-12 종목 교체/커스터마이징 사용자 매뉴얼

## 1. 개요

종목 교체/커스터마이징 기능은 대시보드의 각 위젯에서 표시할 종목을 사용자가 직접 선택/해제할 수 있게 해준다. 불필요한 종목을 숨기고 관심 있는 종목만 표시하여 대시보드를 개인화할 수 있다.

## 3. 사용 방법

### 3.1 설정 모달 열기

1. 대시보드에서 원하는 위젯(국내주식/해외주식/암호화폐)을 찾는다
2. 위젯 제목 옆의 기어 아이콘(⚙)을 클릭한다
3. 종목 설정 모달이 화면 중앙에 나타난다

### 3.2 종목 표시/숨김 설정

1. 모달에서 각 종목 옆의 체크박스를 확인한다
   - 체크(✓) 상태: 해당 종목이 위젯에 표시됨
   - 미체크 상태: 해당 종목이 위젯에서 숨겨짐
2. 체크박스를 클릭하여 표시/숨김을 전환한다
3. 변경은 즉시 적용된다 (위젯에 바로 반영)

### 3.3 기본값으로 초기화

1. 모달 하단 좌측의 "초기화" 버튼을 클릭한다
2. 해당 위젯의 모든 종목이 표시 상태로 복원된다

### 3.4 모달 닫기

다음 3가지 방법으로 모달을 닫을 수 있다:
- 모달 하단 우측의 "확인" 버튼 클릭
- 모달 우측 상단의 닫기(X) 버튼 클릭
- 모달 바깥의 어두운 영역 클릭

## 4. 설정 저장

- 설정은 브라우저의 localStorage에 자동 저장된다
- 브라우저를 닫았다 다시 열어도 설정이 유지된다
- 각 위젯의 설정은 독립적으로 관리된다 (국내주식 설정 변경이 해외주식에 영향 없음)
- 브라우저 데이터를 초기화하면 설정이 기본값으로 복원된다

## 5. 참고 사항

### 5.1 모든 종목을 숨긴 경우

- 모든 종목의 체크를 해제하면 위젯에 다음 안내 메시지가 표시된다:
  > "표시할 종목이 없습니다. 설정에서 종목을 추가하세요."
- 기어 아이콘을 다시 클릭하여 원하는 종목을 활성화할 수 있다

### 5.2 데이터 갱신

- 종목 설정 변경 시 위젯 데이터가 즉시 필터링되어 표시된다
- API는 전체 종목 데이터를 조회하므로 종목을 다시 활성화하면 즉시 데이터가 표시된다

### 5.3 시크릿 모드 / 개인정보 보호 모드

- 시크릿 모드에서는 localStorage가 세션 종료 시 삭제되므로, 설정은 해당 세션 동안만 유지된다
- 기본값(전체 종목 표시)으로 자동 시작된다
