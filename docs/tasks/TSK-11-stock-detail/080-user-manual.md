# TSK-11 종목 상세정보 패널 사용자 매뉴얼

## 1. 개요

종목 상세정보 패널은 국내/해외 주요 종목의 심층 시세 정보를 한눈에 확인할 수 있는 대시보드 위젯이다. 기본 시세 외에 52주 최고/최저, 시가총액 등 투자 판단에 필요한 세부 데이터를 제공한다.

## 2. 대상 종목

| 종목코드 | 종목명 | 시장 | 통화 |
|----------|--------|------|------|
| 058610 | 에스피지 | KOSPI | KRW |
| 247540 | 에코프로 | KOSDAQ | KRW |
| 068270 | 셀트리온 | KOSPI | KRW |
| GOOG | Alphabet C | NASDAQ | USD |
| FIGM | Figma | NYSE | USD |

## 3. 표시 정보

각 종목의 상세 패널에 다음 정보가 표시된다:

| 항목 | 설명 | 예시 |
|------|------|------|
| 현재가 | 실시간 가격 | ₩35,000 / $178.50 |
| 등락금액 | 전일 대비 변동 금액 | +₩500 / -$2.30 |
| 등락률 | 전일 대비 변동 퍼센트 | +1.45% / -1.27% |
| 시가 | 당일 시가 | ₩34,800 |
| 고가 | 당일 고가 | ₩35,500 |
| 저가 | 당일 저가 | ₩34,500 |
| 전일종가 | 전일 종가 | ₩34,500 |
| 거래량 | 당일 거래량 | 123,456 |
| 52주 최고 | 최근 52주간 최고가 | ₩42,000 |
| 52주 최저 | 최근 52주간 최저가 | ₩28,000 |
| 시가총액 | 기업 시가총액 | 5,000억 (데이터가 있는 경우만 표시) |

## 4. 사용 방법

### 4.1 종목 선택

- 패널 상단에 5개 종목 버튼이 표시된다
- 원하는 종목 버튼을 클릭하면 해당 종목의 상세 정보가 표시된다
- 현재 선택된 종목은 진한 색상(검정 배경, 흰색 텍스트)으로 구분된다
- 기본 선택 종목은 에스피지(058610)이다

### 4.2 통화 표시

- 국내 종목(에스피지, 에코프로, 셀트리온): 원화(₩) 형식으로 표시
- 해외 종목(Alphabet C, Figma): 달러($) 형식으로 표시
- 종목 선택 시 통화가 자동으로 전환된다

### 4.3 색상 규칙

| 상태 | 색상 | 의미 |
|------|------|------|
| 상승 (빨간색) | 전일 종가 대비 상승 | 양(+) 변동 |
| 하락 (파란색) | 전일 종가 대비 하락 | 음(-) 변동 |
| 보합 (회색) | 전일 종가 대비 변동 없음 | 0 변동 |

## 5. 데이터 갱신

- 종목 전환 시 자동으로 데이터 조회
- 이전에 조회한 종목은 캐시에서 즉시 표시 후 백그라운드에서 갱신
- 데이터 소스: Yahoo Finance

## 6. 에러 상황

| 상황 | 표시 내용 |
|------|----------|
| 데이터 로딩 중 | 로딩 스피너 표시 |
| 데이터 조회 실패 | "데이터를 불러올 수 없습니다" 메시지 + 다시 시도 버튼 |
| 시가총액 정보 없음 | 시가총액 행 미표시 (나머지 항목은 정상 표시) |

## 7. 위치

대시보드 화면에서 차트 아래 좌측 영역에 4컬럼 너비로 배치된다. 데스크톱 기준 12컬럼 그리드의 좌측 4컬럼을 차지하며, 모바일에서는 전체 너비로 표시된다.
