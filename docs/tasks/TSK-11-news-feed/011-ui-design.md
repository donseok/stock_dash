## 3. 컴포넌트 상세

### 3.1 NewsCard

| 요소 | 설명 | 스타일 |
|------|------|--------|
| 썸네일 이미지 | 뉴스 이미지 | `w-20 h-20 rounded-md object-cover` |
| 제목 | 뉴스 제목 | `text-sm font-medium line-clamp-2` |
| 요약 | 뉴스 설명 | `text-xs text-gray-500 line-clamp-2 mt-1` |
| 출처 | 언론사명 | `text-xs text-gray-400` |
| 발행 시간 | 상대 시간 | `text-xs text-gray-400` |
| 구분선 | 카드 간 | `border-b border-gray-100` |

### 3.2 NewsCard 레이아웃

```
┌──────────────────────────────────────────┐
│ ┌──────┐                                 │
│ │      │  제목 (최대 2줄)                 │
│ │ 이미지│  요약 (최대 2줄)                 │
│ │      │  출처 · 시간                     │
│ └──────┘                                 │
└──────────────────────────────────────────┘
```

- 이미지와 텍스트: `flex gap-3`
- 이미지 없는 경우: 기본 placeholder 이미지 표시
- 카드 패딩: `py-3`

### 3.3 시간 표시 형식

| 경과 시간 | 표시 형식 | 예시 |
|----------|---------|------|
| < 1분 | "방금 전" | 방금 전 |
| < 1시간 | "N분 전" | 15분 전 |
| < 24시간 | "N시간 전" | 3시간 전 |
| < 7일 | "N일 전" | 2일 전 |
| >= 7일 | 날짜 표시 | 2026.01.26 |

### 3.4 Mock 데이터 표시

```
┌────────────────────────────────────────┐
│  ℹ️ 현재 샘플 뉴스가 표시되고 있습니다.  │
└────────────────────────────────────────┘
```

- source_type이 "mock"인 경우 위젯 상단에 안내 배너 표시
- `bg-yellow-50 text-yellow-700 text-xs px-3 py-1 rounded`

## 4. 색상 체계

| 요소 | 색상 | 비고 |
|------|------|------|
| 제목 텍스트 | `text-gray-900` | 기본 |
| 제목 호버 | `text-indigo-600` | 링크 강조 |
| 요약 텍스트 | `text-gray-500` | 부가 정보 |
| 출처/시간 | `text-gray-400` | 메타 정보 |
| 카드 호버 배경 | `bg-gray-50` | 상호작용 피드백 |

## 5. 반응형 디자인

| 화면 크기 | 레이아웃 | 이미지 | 비고 |
|-----------|---------|--------|------|
| Desktop (>=1024px) | 리스트형 | 80x80px | 전체 표시 |
| Tablet (768-1023px) | 리스트형 | 60x60px | 요약 축소 |
| Mobile (<768px) | 리스트형 | 숨김 | 제목 + 출처만 |

## 6. 인터랙션

### 6.1 뉴스 클릭

- 제목 또는 카드 전체 클릭 시 새 탭에서 원문 URL 열기
- `target="_blank" rel="noopener noreferrer"`

### 6.2 자동 갱신

- 5분 간격 자동 갱신
- 새 뉴스 수신 시 부드러운 추가 애니메이션

### 6.3 더 보기

- "더 보기" 버튼 클릭 시 추가 뉴스 로드
- 또는 뉴스 전체 페이지로 이동

## 7. 접근성 (A11y)

| 항목 | 구현 |
|------|------|
| 스크린 리더 | 각 뉴스에 `aria-label="삼성전자 기사, 한국경제, 2시간 전"` |
| 키보드 | Tab으로 뉴스 항목 이동, Enter로 원문 열기 |
| 이미지 | `alt` 속성에 뉴스 제목 설정 |
| 새 탭 | "새 탭에서 열림" 알림 (`aria-label` 보완) |
