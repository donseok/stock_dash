# TSK-11 뉴스 피드 사용자 매뉴얼

## 1. 개요

뉴스 위젯은 금융/주식 관련 최신 뉴스를 대시보드에 표시한다.

## 2. 표시 정보

각 뉴스 카드에 다음 정보가 표시된다:
- **뉴스 제목**: 기사 제목 (최대 2줄)
- **요약**: 기사 요약 내용
- **출처**: 언론사명 (예: 한국경제, 매일경제)
- **발행 시간**: 상대 시간 표시 (예: 3시간 전)

## 3. 기능

### 3.1 뉴스 원문 보기
- 뉴스 카드 클릭 시 새 탭에서 원문 기사 열기

### 3.2 자동 갱신
- 5분 간격으로 최신 뉴스 자동 갱신

### 3.3 새로고침
- 위젯 우측 상단 새로고침 버튼 클릭

## 4. 데이터 소스

| 소스 | 조건 | 표시 |
|------|------|------|
| NewsAPI (실시간) | API 키 설정 및 정상 동작 시 | "live" |
| Mock 데이터 | API 키 미설정 또는 장애 시 | "mock" 안내 배너 표시 |

## 5. 뉴스 카테고리

- 기본 카테고리: 비즈니스 (business)
- 기본 국가: 한국 (kr)
- 기본 표시 수: 10개

## 6. 주의사항

- NewsAPI 무료 플랜은 요청 수 제한이 있음
- API 키 미설정 시 샘플 뉴스가 표시됨
- Mock 데이터는 실시간 뉴스가 아님을 유의
