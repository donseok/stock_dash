# TSK-08 귀금속 사용자 매뉴얼

## 1. 개요

귀금속 위젯은 금(Gold)과 은(Silver)의 국제 현물가를 표시한다.

## 2. 표시 귀금속

| 심볼 | 귀금속 | 단위 |
|------|--------|------|
| XAU | Gold (금) | 트로이온스 (oz) |
| XAG | Silver (은) | 트로이온스 (oz) |

## 3. 표시 정보

- **귀금속명 / 원소기호**: Gold (Au)
- **현재가 (USD)**: 달러 표시 (예: $2,815.50 /oz)
- **현재가 (KRW)**: 원화 환산가 (환율 연동)
- **등락 정보**: 전일 대비 변동금액 및 변동률
- **온스당/그램당 가격**: 단위별 가격 표시

## 4. 색상 규칙

국제 시장 관례에 따라:
- **상승 (초록색)**: 전일 대비 상승
- **하락 (빨간색)**: 전일 대비 하락

## 5. 데이터 갱신

- 자동 갱신 주기: 60초
- 데이터 소스: metals.live API
- API 장애 시 캐시된 데이터 표시

## 6. KRW 환산

- TSK-09 환율 데이터를 활용하여 KRW 환산가 표시
- 환율 데이터 미사용 시 USD만 표시
