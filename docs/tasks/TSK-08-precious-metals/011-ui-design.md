## 3. 컴포넌트 상세

### 3.1 MetalCard

| 요소 | 설명 | 스타일 |
|------|------|--------|
| 원소 기호 배지 | Au / Ag | Gold: `bg-yellow-100 text-yellow-700`, Silver: `bg-gray-200 text-gray-600` |
| 귀금속명 | Gold (금) / Silver (은) | `text-base font-medium` |
| USD 가격 | 달러 기준 가격 | `text-xl font-bold` |
| 단위 | /oz (트로이온스) | `text-sm text-gray-400` |
| 등락 | 전일 대비 변동 | 상승: `text-green-500`, 하락: `text-red-500` |
| KRW 환산가 | 원화 환산 | `text-sm text-gray-600` |

### 3.2 원소 기호 배지

```
Gold:
┌──────┐
│  Au  │  bg-yellow-100 text-yellow-700 font-bold
└──────┘  rounded-md w-10 h-10 flex items-center justify-center

Silver:
┌──────┐
│  Ag  │  bg-gray-200 text-gray-600 font-bold
└──────┘  rounded-md w-10 h-10 flex items-center justify-center
```

### 3.3 가격 표시 형식

| 귀금속 | USD 형식 | KRW 형식 |
|--------|---------|---------|
| Gold | $2,045.50 /oz | ₩2,698,000 /oz |
| Silver | $23.15 /oz | ₩30,500 /oz |

## 4. 색상 체계

| 상태 | 텍스트 색상 | 비고 |
|------|-----------|------|
| 상승 | `#22c55e` (green-500) | 국제 시장 기준 |
| 하락 | `#ef4444` (red-500) | 국제 시장 기준 |
| 보합 | `#6b7280` (gray-500) | 변동 없음 |
| Gold 테마 | `#ca8a04` (yellow-700) | 금 관련 강조 |
| Silver 테마 | `#4b5563` (gray-600) | 은 관련 강조 |

## 5. 반응형 디자인

| 화면 크기 | 레이아웃 | 비고 |
|-----------|---------|------|
| Desktop (>=1024px) | 세로 2개 카드 | 전체 너비 |
| Tablet (768-1023px) | 세로 2개 카드 | 전체 너비 |
| Mobile (<768px) | 세로 2개 카드 | 패딩 축소, KRW 환산가 축약 |
