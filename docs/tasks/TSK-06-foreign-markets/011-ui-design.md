## 3. 컴포넌트 상세

### 3.1 ForeignStockWidget - StockCard

| 요소 | 설명 | 스타일 |
|------|------|--------|
| 종목명 | 영문 종목명 | `text-base font-medium` |
| 심볼 | 티커 심볼 | `text-sm text-gray-500` |
| 거래소 | NASDAQ 등 | `text-xs px-2 py-0.5 rounded bg-blue-50 text-blue-700` |
| 현재가 | USD 표시 ($) | `text-xl font-bold` |
| 등락 | 전일 대비 | 상승: `text-green-500`, 하락: `text-red-500` |
| 거래량 | Volume | `text-sm text-gray-500` |

> 해외 주식은 국제 표준 색상을 사용한다: 상승=초록, 하락=빨강

### 3.2 MarketIndicesBar - IndexBadge

| 요소 | 설명 | 스타일 |
|------|------|--------|
| 지수명 | DOW, NASDAQ | `text-sm font-semibold` |
| 지수값 | 현재 지수 | `text-base font-bold` |
| 등락 | 전일 대비 | 상승: `text-green-500`, 하락: `text-red-500` |
| 등락률 | 퍼센트 | 등락과 동일 색상 |

### 3.3 IndexBadge 레이아웃

```
┌─────────────────────────────┐
│  DOW                        │
│  44,250.50  ▲+125.30(+0.28%)│
└─────────────────────────────┘
```

- 배경: `bg-white border border-gray-200 rounded-md px-3 py-2`
- 호버: `hover:bg-gray-50`

## 4. 색상 체계 (해외 시장)

| 상태 | 텍스트 색상 | 비고 |
|------|-----------|------|
| 상승 | `#22c55e` (green-500) | 국제 표준 상승=초록 |
| 하락 | `#ef4444` (red-500) | 국제 표준 하락=빨강 |
| 보합 | `#6b7280` (gray-500) | 변동 없음 |

> 국내 주식(TSK-05)과 해외 주식의 색상 규칙이 다름에 유의

## 5. 반응형 디자인

| 화면 크기 | ForeignStockWidget | MarketIndicesBar |
|-----------|-------------------|------------------|
| Desktop (>=1024px) | 전체 너비 카드 | 가로 배치 |
| Tablet (768-1023px) | 전체 너비 카드 | 가로 배치 |
| Mobile (<768px) | 전체 너비, 패딩 축소 | 세로 배치 또는 가로 스크롤 |

## 6. 인터랙션

### 6.1 시간대 표시

- 현지 시간과 한국 시간 동시 표시
- 장 운영 상태 표시: "장중" / "장 마감" / "장전"

### 6.2 카드 클릭

- ForeignStockWidget 카드 클릭 시 차트 위젯 연동
- MarketIndicesBar 배지 클릭 시 해당 지수 상세 표시

### 6.3 로딩/에러 상태

- Skeleton 로딩 표시
- 에러 시 "데이터를 불러올 수 없습니다" + 재시도 버튼

## 7. 접근성 (A11y)

| 항목 | 구현 |
|------|------|
| 스크린 리더 | `aria-label="Alphabet C $178.25 up $2.15"` |
| 키보드 | Tab으로 카드/배지 이동 |
| 색상 대비 | WCAG 2.1 AA 기준 충족 |
