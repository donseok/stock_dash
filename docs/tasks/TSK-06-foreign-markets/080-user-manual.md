# TSK-06 해외 주식 사용자 매뉴얼

## 1. 개요

해외 주식 위젯은 미국 주식시장(NYSE/NASDAQ)의 주요 종목 시세를 표시한다.

## 2. 표시 종목

| 종목코드 | 종목명 | 시장 |
|----------|--------|------|
| GOOGL | Alphabet (Google) | NASDAQ |

## 3. 표시 정보

- **종목명 / 티커**: Alphabet (GOOGL)
- **현재가**: USD 표시 (예: $175.50)
- **전일 대비**: 등락금액 및 등락률
- **거래량**: 당일 거래량

## 4. 색상 규칙

국제 시장 관례에 따라:
- **상승 (초록색)**: 전일 종가 대비 상승
- **하락 (빨간색)**: 전일 종가 대비 하락

## 5. 데이터 갱신

- 자동 갱신 주기: 30초
- 데이터 소스: Yahoo Finance
- 미국 시장 마감 시 마지막 종가 표시
