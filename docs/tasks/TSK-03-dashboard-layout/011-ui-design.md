# TSK-03: 대시보드 레이아웃 - UI 설계

## 1. 화면 구성

### 데스크톱 (>= 1024px)

```
+-------------------------------------------+
| [Logo] Stock Dashboard   2026년 2월 2일 (일) |  ← Header (h-14)
+-------------------------------------------+
| KOSPI 2,580 +0.48% | KOSDAQ 742 -0.43% | DOW ... |  ← MarketIndicesBar (h-10)
+-------------------------------------------+
| 주식 차트 (8col)         | 환율 (4col)      |
|                          | 귀금속 (4col)    |
+-------------------------------------------+
| 국내주식 (6col)          | 해외주식 (6col)  |
+-------------------------------------------+
| 암호화폐 (8col)          | 뉴스 (4col)     |
+-------------------------------------------+
```

## 2. 컴포넌트 상세

### Header
- 높이: 56px (h-14)
- 로고 + 앱 이름 좌측
- 날짜 + 연결 상태 표시 우측
- `sticky top-0 z-50`

### MarketIndicesBar
- 높이: 40px (h-10)
- 어두운 배경 (bg-gray-900)
- 지수명, 현재값, 등락률 표시
- 가로 스크롤

### Card 위젯
- 흰색 배경, border, rounded-lg
- 타이틀 바 + 콘텐츠 영역
- shadow-sm

## 3. 상태 표시

| 상태 | UI |
|------|-----|
| 로딩 | Spinner 애니메이션 |
| 에러 | 에러 메시지 + 재시도 버튼 |
| 데이터 없음 | 안내 메시지 |
| 가격 상승 | 빨간색 + ▲ |
| 가격 하락 | 파란색 + ▼ |
