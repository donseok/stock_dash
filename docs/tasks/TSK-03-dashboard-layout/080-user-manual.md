# TSK-03 대시보드 레이아웃 사용자 매뉴얼

## 1. 개요

대시보드는 stock_dash 웹 포털의 메인 화면으로, 시장 데이터를 한눈에 파악할 수 있는 위젯 기반 레이아웃을 제공한다.

## 2. 화면 구성

### 2.1 헤더 영역
- 앱 타이틀 "Stock Dashboard" 표시
- 현재 날짜 및 시간 표시
- 시장 개장/마감 상태 표시

### 2.2 마켓 인덱스 바
- 화면 상단에 주요 지수(KOSPI, KOSDAQ, DOW, NASDAQ) 티커 표시
- 실시간 등락 정보 업데이트

### 2.3 위젯 그리드
- 12컬럼 그리드 시스템 기반 반응형 레이아웃
- 국내 주식, 해외 주식, 암호화폐, 귀금속, 환율, 뉴스 위젯 배치
- 주식 차트 위젯은 전체 너비 사용

## 3. 반응형 동작

| 화면 크기 | 동작 |
|-----------|------|
| Desktop (≥1280px) | 3컬럼 그리드, 모든 위젯 표시 |
| Tablet (768-1279px) | 2컬럼 그리드 |
| Mobile (<768px) | 1컬럼 스택 |

## 4. 접근 방법

1. 브라우저에서 `http://localhost:3000` 접속
2. 대시보드가 자동으로 로드됨
3. 각 위젯은 독립적으로 데이터를 갱신함
