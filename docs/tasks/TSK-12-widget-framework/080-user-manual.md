# TSK-12 위젯 프레임워크 사용자 매뉴얼

## 1. 개요

위젯 프레임워크는 stock_dash 대시보드의 모든 위젯이 사용하는 공통 UI 컴포넌트, 데이터 페칭 패턴, 유틸리티 함수를 제공한다.

## 2. 공통 컴포넌트

### 2.1 Card
모든 위젯의 기본 컨테이너. 타이틀과 본문 영역을 포함한다.

### 2.2 LoadingSpinner
데이터 로딩 중 표시되는 회전 애니메이션. 소형(sm), 중형(md), 대형(lg) 크기를 지원한다.

### 2.3 ErrorMessage
데이터 로딩 실패 시 표시. "다시 시도" 버튼을 포함한다.

### 2.4 PriceChange
가격 변동 정보를 화살표와 색상으로 표시. 시장 유형(국내/해외)에 따라 색상 규칙이 다르다.

## 3. 데이터 갱신 방식

각 위젯은 TanStack Query를 통해 자동으로 데이터를 갱신한다:

| 위젯 | 갱신 주기 |
|------|----------|
| 시장지수 | 10초 |
| 암호화폐 | 10초 |
| 국내/해외 주식 | 30초 |
| 귀금속 | 60초 |
| 환율 | 60초 |
| 뉴스 | 300초 |

## 4. 가격 표시 형식

- **KRW**: ₩79,000 (천 단위 콤마)
- **USD**: $175.50 (소수점 2자리)
- **등락률**: +1.54% (부호 + 소수점 2자리)
- **거래량**: 15,234,567 (천 단위 콤마)
