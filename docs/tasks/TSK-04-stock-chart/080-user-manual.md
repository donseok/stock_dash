# TSK-04 주식 차트 사용자 매뉴얼

## 1. 개요

주식 차트 위젯은 TradingView Lightweight Charts를 활용하여 종목의 OHLC(시가/고가/저가/종가) 캔들스틱 차트를 제공한다.

## 2. 기능 설명

### 2.1 종목 선택
- 차트 상단의 종목 코드 버튼을 클릭하여 변경
- 기본 종목: 삼성전자 (005930)
- 지원 종목: 국내 주식 (005930, 247540, 068270), 해외 주식 (GOOGL)

### 2.2 기간 선택
- 1D (1일), 1W (1주), 1M (1개월), 3M (3개월), 1Y (1년) 선택 가능
- 기간 버튼 클릭 시 즉시 차트 갱신

### 2.3 차트 조작
- 마우스 드래그: 차트 이동
- 마우스 휠: 확대/축소
- 십자선(크로스헤어): 마우스 위치의 가격/시간 표시

## 3. 데이터 갱신

- 자동 갱신 주기: 30초
- 수동 갱신: 새로고침 버튼 클릭

## 4. 주의사항

- Yahoo Finance 데이터 소스 사용으로 15분 지연 가능
- 시장 마감 시간에는 당일 마지막 데이터까지 표시
